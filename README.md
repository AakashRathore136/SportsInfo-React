### **SportsInfo**

A frontend web application built using React that displays sports-related information through a clean and responsive user interface.

---

## Overview

SportsInfo is a client-side application focused on building and structuring UI components using React.
The project demonstrates component-based architecture, state management, and responsive design without relying on external APIs or backend services.

---

## Features

* Component-based architecture using React
* Clean and structured UI
* Responsive layout for different screen sizes
* Reusable components
* Simple and maintainable code structure

---

## Tech Stack

* React
* JavaScript (ES6+)
* HTML5
* CSS3
